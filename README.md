# python-cloud

## Install dependencies
```
pip install -r requirements.txt
```

Prepare venv:
```
python3 -m venv venv
```

Run virtual env:
```
. venv/bin/activate
```

Run application:
```
flask run
```

Run dev enviroment:
```
export FLASK_ENV=development
```

Init database:
```
flask init-db
```
